Bekannte technische Hinweise:
- Auf ARM64/Termux-Proot muss Gradle AAPT2 ueber den qemu-Wrapper laufen:
  - `-Pandroid.aapt2FromMavenOverride=/tmp/aapt2`
  - Das Build-Script erkennt `/tmp/aapt2` automatisch.
- Android toybox `tar` hatte bei frueheren Rootfs-Seeds fehlerhafte Archive erzeugt (`bad header` / EOF). Loesung: Export vom Android-Geraet, dann lokales Repacking mit GNU tar und xz.
- Die APK kann nicht direkt als normale Git-Datei zu GitHub gepusht werden, weil sie ueber 100 MB gross ist. Verwende GitHub Releases oder Git LFS. In dieser Session wurde ein Release-Asset verwendet.
- Der GitHub-Release-Tag lautet `android-delivery-proot-voxtral-debug`.
