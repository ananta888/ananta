Ananta

AI Agent Management System für Softwareentwicklung.

Ananta soll KI-gestützte Entwicklung kontrollierbarer machen: mit Hub, Workern, klaren Policies, nachvollziehbaren Tasks und begrenztem Zugriff auf Projektwissen.

GitHub ansehen Was es zeigen soll

Agenten-Steuerung

Zentrale Verwaltung von Workern, Rollen, Laufumgebungen und Ausführungsmodi.

Kontrollierter Kontext

Projektwissen soll gezielt an passende Worker gegeben werden, nicht blind alles an jedes Modell.

Nachvollziehbare Tasks

Planung, Ausführung, Artefakte, Tests und Review sollen sichtbar und prüfbar bleiben.
