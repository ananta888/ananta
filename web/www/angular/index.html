Angular-Projekt (Präsentation)

Das Angular-Subprojekt ist das Bedien- und Observability-Frontend von Ananta.

Diese Seite beschreibt das Angular-Teilprojekt als Produktpräsentation. Sie liefert bewusst keine gebaute Angular-App aus, sondern zeigt Architektur, Umfang und Nutzen.

Frontend im Repo CodeCompass ansehen

Dashboard

Live-Übersicht über Goals, Tasks, Worker-Status, Laufzeit-Events und Artefakte.

Auto-Planner

Aus Zielen werden nachvollziehbare Task-Pläne mit Abhängigkeiten und Ausführungsstatus.

Terminal & Runtime

Operator-Ansicht für Befehle, Runs, Logs und kontrollierte Agenten-Ausführung.

Voice / Offline

Geplante bzw. integrierte Sprach-UX über Voxtral-Ansatz für lokale Bedienung.

Android via Capacitor

Dasselbe Frontend läuft als mobile App für unterwegs, inklusive API-Anbindung.
