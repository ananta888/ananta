# Licensing Overview (English)

Ananta is provided under **dual licensing**:

- **Open Source:** GNU Affero General Public License v3.0 or later (**AGPL-3.0-or-later**)
- **Commercial:** available from the copyright holder under separate terms

## Why AGPL?

Ananta is a hub-worker, API-driven, multi-agent platform that can also be used as a hosted or internal network service.
The AGPL helps ensure that improvements to the software remain available to users even when the software is used over a network.

## What AGPL means in practice

You may:

- use the software privately or internally
- study and modify the source code
- redistribute original or modified versions

If you modify Ananta and make it available to users over a network, the AGPL generally requires you to provide the corresponding source code of that modified version to those users.

## Commercial licensing

If you want to use Ananta under different terms, for example:

- in proprietary products
- in closed-source SaaS offerings
- with negotiated warranty, support, or OEM terms

then a separate commercial license is required.
