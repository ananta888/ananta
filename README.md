# ananta

Dies ist eine Beispielanwendung mit Java Backend (Spring Boot) und Angular Frontend zum Verwalten von Kursen und Videos.

## Backend starten

```bash
cd backend
mvn spring-boot:run
```

## Frontend starten

```bash
cd frontend
npm install
npm start
```

Für einen Produktions-Build des Frontends kann anschließend ausgeführt werden:

```bash
npx ng build
```
