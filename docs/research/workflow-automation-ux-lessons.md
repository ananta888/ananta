# Workflow Automation UX Lessons

Date: 2026-04-30

Patterns worth adopting:
- Visible execution history and retry semantics.
- Explicit connector-level credential boundaries.
- Human approval nodes for risky transitions.
- Clear trigger and callback observability.

Boundaries to keep:
- Do not move planning/governance ownership out of Hub.
- Do not mirror external engine UIs as core Ananta architecture.
