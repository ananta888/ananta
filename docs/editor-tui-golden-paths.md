# Golden Paths: Plugin and TUI

## Golden path: Neovim plugin

1. Configure profile and authenticate.
2. Analyze current file with `AnantaAnalyze`.
3. Review selected block with `AnantaReview`.
4. Submit follow-up goal with `AnantaGoalSubmit`.
5. Inspect task/artifact context and open deep details in browser if needed.

## Golden path: TUI

1. Open TUI and confirm profile/health header.
2. Inspect task board and one task detail.
3. Inspect related artifact and recent logs.
4. Review one approval queue item and complete explicit decision flow.
5. Verify audit summary and diagnostics/KRITIS visibility before finishing.
