## Public Positioning

Ananta should be described as a controlled orchestration system, not as a generic chatbot or unconstrained automation runner. Public wording should emphasize:

- hub-owned planning and task control
- worker execution through delegated tasks
- auditable verification and artifacts
- explicit release and security governance

Avoid claims that imply autonomous production deployment, unsupervised code execution or worker-to-worker orchestration.
