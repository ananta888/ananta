# Workflow Automation Runtime Profile

Date: 2026-04-30

Workflow automation is optional.

- Default: disabled.
- Recommended first mode: dry-run with mock provider.
- Optional config knobs:
  - `WORKFLOW_AUTOMATION_ENABLED`
  - `WORKFLOW_PROVIDER_ENDPOINT`
  - `WORKFLOW_CALLBACK_SECRET`

No workflow engine credential is required for baseline local Ananta usage.
