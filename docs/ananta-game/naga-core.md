# NagaCore

NagaCore ist Guide und Visualisierungsschicht des Strategy Games, nicht Security- oder Policy-Authority.

## Rolle

- erklaert Spiel-/Architekturkonzepte in Tutorial-Form
- verbindet Snake-Identitaet mit Ananta-Metapher
- liefert UI-neutrale Step-Payloads fuer TUI/Web

## Tutorial-Themen

1. **AegisHub**: zentrale Hub-Kontrolle und Delegation
2. **ContextAegis**: Kontextgrenzen und Default-Deny
3. **ArtifactGuard**: Evidence statt behaupteter Completion
4. **CodeCompass**: Karten-/Kontextgrundlage

## Sicherheitsgrenze

NagaCore trifft keine policy-kritischen Entscheidungen, sondern erklaert sie nur.
