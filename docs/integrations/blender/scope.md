# Blender Integration Scope

This track implements Blender as a thin Ananta control surface.

## In Scope
- Read-only scene inspection and bounded context capture
- Goal submission to Hub
- Task/artifact/approval visualization
- Approval-gated execution bridge for mutating operations

## Out of Scope
- FreeCAD/KiCad integration
- Addon-owned orchestration/policy loops
- Unbounded scene/source/doc ingestion
