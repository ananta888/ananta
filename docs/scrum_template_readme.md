# Scrum Team Template

Dieses Template bietet eine vollständige Struktur für ein Scrum-Team innerhalb der Ananta-Plattform.

## Enthaltene Artefakte

- **Backlog**: Ein zentraler Ort für alle User Stories und Anforderungen.
- **Sprint Board**: Visualisierung des aktuellen Arbeitsfortschritts (To-Do, In-Progress, Done).
- **Burndown Chart**: Verfolgung des Fortschritts innerhalb eines Sprints.
- **Roadmap**: Visualisierung von langfristigen Meilensteinen und Zielen.
- **Instruktionen**: Eine detaillierte Anleitung zur Nutzung des Templates.

## Einrichtung

1. Gehen Sie in der Anwendung auf den Bereich **Teams**.
2. Klicken Sie auf **Team anlegen**.
3. Geben Sie einen Namen ein und wählen Sie als Typ **Scrum** aus.
4. Nach dem Speichern werden automatisch alle Artefakte als Aufgaben (Tasks) für dieses Team angelegt.
5. Fügen Sie Ihre Teammitglieder (Agenten) zum Team hinzu.

## Nutzung

- Verwenden Sie das **Board**, um Aufgaben zu verschieben.
- Pflegen Sie das **Backlog** regelmäßig.
- Folgen Sie den Schritten im Task **Setup & Usage Instructions** für die Integration mit externen Systemen wie GitHub.
